Rate Limiting (Continued)

Rate limit information is included in the response headers:

X-RateLimit-Limit: 1000 X-RateLimit-Remaining: 999 X-RateLimit-Reset: 1623967345

If you exceed the rate limit, you'll receive a 429 Too Many Requests response.

Webhooks

RIELEC supports webhooks for real-time notifications of events in your account.

POST /webhooks

Create a new webhook subscription.

Request Body:

{ "url": "https://your-server.com/webhook", "events": ["instance.created", "instance.deleted", "instance.started", "instance.stopped"], "secret": "your_webhook_secret" }

Response:

{ "id": "whk-123abc", "url": "https://your-server.com/webhook", "events": ["instance.created", "instance.deleted", "instance.started", "instance.stopped"], "created_at": "2023-06-17T16:00:00Z" }

Auto-scaling

POST /instance-groups/{group_id}/auto-scaling

Configure auto-scaling for an instance group.

Request Body:

{ "min_instances": 2, "max_instances": 10, "target_cpu_utilization": 70, "cooldown_period": 300 }

Response:

{ "group_id": "grp-789ghi", "auto_scaling": { "min_instances": 2, "max_instances": 10, "target_cpu_utilization": 70, "cooldown_period": 300, "enabled": true } }

Logs

GET /instances/{instance_id}/logs

Retrieve logs for a specific instance.

Query Parameters:

ParameterTypeDescription
start_timestringStart time for logs (ISO 8601 format)
end_timestringEnd time for logs (ISO 8601 format)
limitintegerMaximum number of log entries to return (default: 100, max: 1000)

Response:

{ "instance_id": "inst-123abc", "logs": [ { "timestamp": "2023-06-17T10:30:00Z", "level": "INFO", "message": "Application started successfully" }, { "timestamp": "2023-06-17T10:35:15Z", "level": "ERROR", "message": "Database connection failed" }, ... ] }

Networking

POST /networks

Create a new network.

Request Body:

{ "name": "app-network", "cidr": "172.18.0.0/16" }

Response:

{ "id": "net-456def", "name": "app-network", "cidr": "172.18.0.0/16", "created_at": "2023-06-17T17:30:00Z" }

POST /instances/{instance_id}/network

Connect an instance to a network.

Request Body:

{ "network_id": "net-456def" }

Response:

{ "instance_id": "inst-123abc", "network_id": "net-456def", "ip_address": "172.18.0.5" }

Backups

POST /instances/{instance_id}/backups

Create a backup of an instance.

Request Body:

{ "name": "pre-update-backup", "description": "Backup before major update" }

Response:

{ "id": "bkp-789ghi", "instance_id": "inst-123abc", "name": "pre-update-backup", "description": "Backup before major update", "status": "creating", "created_at": "2023-06-17T18:45:00Z" }

POST /instances/{instance_id}/restore

Restore an instance from a backup.

Request Body:

{ "backup_id": "bkp-789ghi" }

Response:

{ "instance_id": "inst-123abc", "backup_id": "bkp-789ghi", "status": "restoring" }

Integrations

Advanced Metrics

GET /instances/{instance_id}/advanced-metrics

Retrieve advanced metrics for a specific instance.

Query Parameters:

ParameterTypeDescription
start_timestringStart time for metrics (ISO 8601 format)
end_timestringEnd time for metrics (ISO 8601 format)
metricsstringComma-separated list of advanced metrics (e.g., disk_iops,network_packets,etc)

Response:

{ "instance_id": "inst-123abc", "metrics": { "disk_iops": [ {"timestamp": "2023-06-17T20:00:00Z", "read": 1500, "write": 500}, {"timestamp": "2023-06-17T20:01:00Z", "read": 1200, "write": 600}, ... ], "network_packets": [ {"timestamp": "2023-06-17T20:00:00Z", "in": 10000, "out": 8000}, {"timestamp": "2023-06-17T20:01:00Z", "in": 12000, "out": 9000}, ... ] } }
